# Conciseness — what improvement looks like across score bands

Progression notes used by the rep-to-rep analyst to ground observations
when a user's conciseness score moves between bands.

## 0–40 — structurally bloated

The rep contains obvious padding. Multiple preambles before getting to the point. Filler rate is very high (> 6/min) or repetition is obvious (the same idea stated 2+ times). Time budget blown through with extra content that adds no signal. Common at this band: the user is warming up into their argument rather than leading with it.

## 40–60 — improving but still loose

The rep has a main point but leaks time. Fillers are present but not dominant. Preambles exist but are shorter. At this band, users are starting to feel where they're being loose — often the feedback lands with "yeah, I know I repeated myself there."

## 60–75 — tightening

Main point lands early. Supporting material advances the argument. Filler rate has dropped into the 2–4/min range. Time budget respected within ~15%. At this band, the user has internalized BLUF openings — the first sentence is the answer.

## 75–90 — concise

Each sentence earns its place. Main point, support, close — clean. Filler rate < 2/min. Time budget within 10%. At this band, the user is making deliberate word choices — often cutting sentences mid-rep rather than restating them.

## 90+ — signal-dense

Pro-level conciseness. The rep lands faster than the listener expected. Every clause carries meaning. Silences are intentional. At this band, the rep ends with time remaining and that feels like confidence, not a shortfall.

## What to coach when conciseness is moving up

- Call out the specific sentence the user cut or tightened
- Point to the moment the main point landed (timestamp helps)
- Note reduction in hedge-stacking if visible

## What to coach when conciseness is stuck

- Identify one recurring padding pattern (preamble, hedge chain, restatement)
- Give a concrete alternative the user could have said instead
- Ask the user to try the same prompt with a 15-second time cap in the next rep
